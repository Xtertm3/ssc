# Hero Slider Images

Place your hero slider images in this folder.

**Required images**: 3 images
- slide-1.jpg (or .png)
- slide-2.jpg (or .png)
- slide-3.jpg (or .png)

**Specifications**:
- Size: 1920 x 1080 pixels
- Format: JPG or PNG
- Theme: Professional financial/business images

After adding images, update src/components/Home.jsx to use them.
